# Media<a name="assets-media"></a>

You can import media files into Sumerian to use as textures or audio objects\. Files up to 10 MB are supported in the following formats\.

**Images**

+ CRN

+ DDS

+ JPG, JPEG

+ PNG

+ SVG

+ TGA

**Sound**

+ OGG

+ MP3

+ WAVE, WAV

**Video**

+ MP4

+ OGV

+ WEBM

**Text**

+ JS

+ JSON
